Exploiter ces vulnérabilités
Repérer les vulnérabilités
Nessus est un scanner de vulnérabilités qui identifie les faiblesses de sécurité dans un système informatique, réseau ou application. Il peut par la suite permettre d'exploiter des vulnérabilités ou de renforcer la sécurité de ces vulnérabilités Nmap est un outil de découverte de réseau et de sécurité qui permet d'analyser des réseaux informatiques en identifiant les hôtes. Il est utilisé pour la cartographie des réseaux, la vérification de sécurité, la gestion des vulnérabilités ou alors pour savoir si une machine est filtrée ou qui contient un firewall. Kali est une distribution Linux spécialisée dans la sécurité informatique et les tests de pénétration. Elle offre un ensemble d'outils préinstallés dédiés à l'analyse de sécurité, la recherche de vulnérabilités Tcpdump est un outil en ligne de commande utilisé pour capturer et analyser les paquets réseau transitant sur un réseau informatique.Il permet aux administrateurs système et aux professionnels de la sécurité de surveiller le trafic réseau
Les Injections SQL sont une forme d'attaque informatique où des données non fiables sont injectées dans des requêtes SQL afin de manipuler la base de données sous-jacente. On peut utiliser cette technique pour modifier une base de données ou avoir des droits admin. Metasploit est un outil permettant les tests et exploits de sécurité. Il peut identifier les vulnérabilités mais il est surtout utilisé pour exploiter les données et tester la sécurité des systèmes informatiques Scapy est un module python permettant d'initier des paquets Réseau tel que TCP/IP. On peut l'utiliser pour réaliser des attaques de DNS Spoofing, ARP Poisoning, Man in the middle ou bien des vols de sessions/TCP reset. Jhon the reaper est un outil de craquage de mot de passe, celui-ci est utilisé pour tester la robustesse des mots de passe en effectuant des attaques de dictionnaire et des attaques BRUT Force. Une attaque DDoS est utilisé pour perturber le fonctionnement normal d'un service en ligne ou d'un réseau informatique en submergeant la cible avec un trafic réseau excessif.

Repérer les vulnérabilités

  • Utilisation du logiciel Nessus pour lancer des scans approfondis sur des machines
  • Lancement de scans nmap pour vérifier l'états de ports/service et versions de protocoles.
  • Exploitation de failles sous KaliLinux car de nombreux modules y sont initialement installés

Exploiter ces vulnérabilités

  • Utilisation de Metasploit pour tester/vérifier les failles trouvées auparant.
  • Créations de programme Scapy (Python) pour réaliser des attaques tel que l'ARP Poisonning, DNS Spoofing, Session Reset ou encore DDOS
  • Usage du logiciel John The Reaper pour cracker des mot de passes par BRUTForce
  • Réalisations d'injections SQL sur un site pour vérifier l'intégrité d'une base de données
