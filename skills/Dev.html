Développement logiciel et système
Bash est un interpréteur de commandes et un langage de script utilisé principalement dans les systèmes d'exploitation de type Unix. Il permet l'automatatisation des tâches grâce à des scripts. SQL est un langage de programmation spécialisé utilisé pour gérer et manipuler des bases de données relationnelles. Il permet de définir, d'interroger, de mettre à jour et de gérer les données stockées Python est un langage de programmation polyvalent orienté objet permettant le développement d'applications web, le traitement de données, l'automatisation de tâches mais aussi la proggramation évènementielle grâce aux Threads Une API est un ensemble de règles et de protocoles permettant à différentes applications de communiquer entre elles, facilitant l'échange de données entre serveurs. Json est un format de données léger et facile à lire, basé sur la syntaxe JavaScript, utilisé pour échanger des informations structurées entre différents langages de programmation ou applications.
