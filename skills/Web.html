Développement web
HTML est un langage de balisage utilisé pour créer et structurer le contenu de pages webs. Chaque page web nécéssite du code HTML. CSS est un langage de style utilisé pour styliser les balises HTML. Il est souvent utilisé pour réaliser des animations tel que les logos que vous voyez en fond Javascript est un langage de programmation de scripts principalement utilisé côté client pour interagir dynamiquement avec le contenu des pages ou manipuler des données. PHP est un langage de script côté serveur, conçu pour le développement web, permettant d'exécuter des opérations sur le serveur, générer du contenu dynamique, et interagir avec des bases de données

Front-End

  • Scripts d'automatisaiton systèmes
  • Scripts de gestions de fichiers et répertoires
  • Scripts de gestion de processus

Back-End

  • Créations de programme Scapy (Python) pour réaliser des attaques tel que l'ARP Poisonning, DNS Spoofing, Session Reset ou encore DDOS
  • Création de programmes permettant d'intéragir avec un utilisateur pour gérer des données
  • Analyse et traitement de données sous forme de fichier CSV/Tableau
  • Développement de mini-jeux
